Listing Advanced Queries
========================

Function
--------

This API is used to list all advanced queries.

Calling Method
--------------

For details, see :ref:`Calling APIs <rms_03_0000>`.

URI
---

GET /v1/resource-manager/domains/{domain_id}/stored-queries
